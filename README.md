# Dancing Alice/Arisu

Dancing Alice Website in 3D :)

NOTE: Turn on music (Bottom Left)

## Installation

```
git clone
```

```
npm install
```

```
npm run dev
```

## Technical Domain

- Vite + React
- React-Three-Fiber for rendering 3D object
